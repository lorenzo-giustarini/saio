# Code of Conduct

This project follows the **[Contributor Covenant 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/)**.

In short: be kind, be respectful, assume good faith. Discrimination, harassment, personal attacks, and similar behavior are not tolerated.

## Reporting

If you experience or witness unacceptable behavior, please report it privately to:

- **Email**: [lorenzo@revolutionmarketing.us](mailto:lorenzo@revolutionmarketing.us)

All reports are reviewed confidentially. Project maintainers will respond as quickly as possible and take appropriate action.

## Scope

This Code of Conduct applies in all project spaces (issues, pull requests, discussions, code comments) and when an individual is officially representing the project in public spaces.
